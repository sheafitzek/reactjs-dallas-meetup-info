This app was developed for a [Call-Em-All](https://www.call-em-all.com/) coding challenge. It hits the [meetup.com](https://www.meetup.com/) v3 api to display info on upcoming events for the [ReactJs-Dallas](https://www.meetup.com/ReactJS-Dallas/) (or any other) Meetup group and the rsvps to those events.
